1 You can use Bootstrap by downloading it or via Content Delivery Network. Bootstrap pages also show you want you want to look for.
2 CDN is content delivery network, which delivers content to users based on their location, the webpage's origin, etc.
3 CDN is quick and easy to use. Content can be displayed to users faster.
4 Breakpoints are points where websites' contents will display the best available layout (for RWDs).
5 The RWD breakpoints use media queries to determine key breakpoints. Media queries vary based on the device with which you're viewing a website, like a tablet, android phone, desktops, etc.
6 The difference between a fixed-width container and a full-width container is that the former only has fixed sizes for devices based on their screen sizes. A full-width container adjusts all of its pixels to take up the available space on the screen, so this is more flexible than a fixed-one.
7 Boostrap has a grid system that breaks a page up into 12 columns.
8 In RWDs, rows give columns a place to exist. Rows and columns go hand in hand, and you can wrap columns in a row. It is also useful for putting constraints on elements horizontally.

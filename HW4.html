  1. The Box Model is a box that goes around elements in HTML, which contains data, margins, padding, etc.
  2. The block level element is something that takes up the whole space for its parent element.
  3. The < p > tag by default is a block-level element.
  4. An inline element is another category of HTML elements besides block-level. Inline is a text-level element, so, they only hold texts and other data unlike blocks (which take up the whole space of their parent elements).
